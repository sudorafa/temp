# Test-Vibbrante

This project is a frontend test to becomes a Vibbrante.
The challenge is make a web application to manage invoices to MEI companies.

## About

MEI-NF is a front-end Angular SPA based to manage invoices and expenses to MEI companies.

## Codebase

### Running the application

1. Clone the repository or download the files
2. Simply running npm install && npx run start app is enough to start.
   ```bash
   npm install && npm start
   ```
